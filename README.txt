# Binary to Text Converter ® by Admir

This application allows you to convert binary data to text and generate images from text descriptions using an API. It features a user-friendly graphical interface with a Matrix-style theme.

## Features
- Convert binary data to text
- Convert text to binary data
- Generate images from text descriptions
- Paste and translate images into binary code
- User-friendly GUI with a dark theme and green text
- Clear and copy functions
- Fullscreen and resizable window
- Quit button
- Custom application icon

## Requirements
- Python 3.x
- Requests library
- Pillow library
- tkinter library

## Installation
1. Run the installer (BinaryToTextConverter_Setup.exe).
2. Follow the on-screen instructions to complete the installation.

## Usage
1. Launch the application from the Start Menu or desktop shortcut.
2. Enter binary data or text to convert.
3. Use the buttons to preview, generate images, or paste images.
4. Use the copy and clear functions as needed.
5. Click "Quit" to close the application.
